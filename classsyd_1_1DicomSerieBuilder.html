Detailed Description

This class search for dicom series in folder and insert the dicom serie in the db. Replace if an element with the same dicom_uid already exists (if SetReplaceSerieFlag(true)). A check on the patient name/id is performed (could be forced with SetForcePatientFlag(true)). In the given serie, filenames is a space separated list of files (without folder)
